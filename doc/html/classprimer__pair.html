Detailed Description

Primer pair design class.

The main class for designing pairs of primers

Having selected the location of the target and/or regions in which the primers may be located, there are three steps :- (1) Generate a number of candidates subject to hard constraints. (2) Analyse and score candidate primers (3) Sort according to score for most optimum pairs.

Hard constraints: primer location, primer length
