A very basic method suitable for short sequences no more than 13 bp. Not recommended, but included for research/historical purposes. Returns the temperature.

double annealing_temperature::Wallace_method ( const char *  sequence)

Non linear equation method based on sequence length and GC content. Does not compensate for GC clustering. Maps primer3's Santa Lucia algorithm well for non clustered sequences and handles sequences > 32 bp. Returns the temperature.
